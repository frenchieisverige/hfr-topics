Tout ce qu'on sait pour l'instant, c'est que la mise à jour sera gratuite pendant un an, depuis une licence valide de Windows 7 ou Windows 8.1
Passé ce délai, il faudra probablement payer pour les retardataires.
Mais pour ceux qui ont fait l'installation à temps, l'utilisation de Windows 10 restera gratuite, passé le délai d'un an.

[b]Que devient ma licence W7/W8 après mise à jour ?[/b]
Une licence OEM W7/W8 devient une licence OEM W10.
Tous ceux qui font la mise à jour gratuite, à partir d'une version BOITE ou OEM, obtiennent la même clé. 
Ensuite, le PC est reconnu sur les serveurs de Microsoft, en fonction de son identification hardware.
Ainsi, il est possible de faire une clean install sur ce PC avant ou après le 29 juillet 2016.

[b]FAQ[/b]
 
[b]Est-ce que je peux faire une clean install sans passer par l'étape "Mise à jour" de mon installation actuelle ?[/b] 
OUI ! On peut faire une clean install uniquement depuis Windows 10 th2 (version 1511).
Par exemple, si la licence d'origine est W7, alors il suffit d'installer W10 en saisissant sa clé W7.
Pour la clé a saisir lors de la réinstallation, si la machine est la même que lors de la Mise à Jour, pas besoin de rentrer de clé. 
 
[b]Windows refuse ma clé 7/8 lors de la réinstallation [/b]
Si la machine a bien été mise à jour depuis 7/8 vers 10 sans changement de materiel depuis, il suffit de cliquer sur "Ignorer cette étape" ou "plus tard" lorsque la clé est demandée lors de la réinstallation.

[b]Quelle différence entre la version 1511 et la 1511.2 ?[/b]
la version 1511 est sortie en novembre 2015 puis la version 1511.2 en avril 2016. 
la version 1511.2 est une version 1511 qui inclut toutes les mises à jour entre novembre 2015 et avril 2016.
